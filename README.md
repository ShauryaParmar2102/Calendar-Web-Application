# Calendar-Web-Application
A simple JavaScript calendar application that allows users to create, view, and delete events for specific days. Events are saved using localStorage so they persist after refresh.

---

## ✨ Features

- 🗓️ Monthly calendar view
  
- ⬅️➡️ Navigate between months
  
- 📝 Add events to specific dates
   
- 👀 View saved events
  
- 🗑️ Delete events
  
- 💾 Persistent storage using localStorage
  
- 🔵 Highlights the current day  

---


## 🛠️ Tech Stack

- 🌐 HTML
  
- 🎨 CSS
  
- ⚙️ JavaScript 

---

## 🚀 How to Run
1. Download the project (or clone it)

2. If downloaded as ZIP, unzip/extract the folder
   
3. Open index.html in your browser

4 .Start using the calendar
